Плоские — содержат один тип данных зато быстро работают, str/bytes например

### Распаковка 
```python 
a, b, *c = [1, 2, 3, 4, 5]

c — [3,4,5]
```

```python 
arr = [1, 2, 3, 4, 5]
arr[2:5] = 100 # Неверно !
```
